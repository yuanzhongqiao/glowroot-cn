<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Glowroot 使用</font></font><a href="https://immutables.github.io" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Immutables</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">注释处理来消除对大量样板代码的维护。</font><font style="vertical-align: inherit;">如果您使用的是 Eclipse，则需要安装</font></font><a href="https://github.com/jbosstools/m2e-apt"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">m2e-apt</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">插件并将“窗口”&gt;“首选项”&gt;“Maven”&gt;“注释处理”更改为“自动配置 JDT APT”。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">要处理 UI，请在您最喜欢的 IDE 中的调试器下运行 org.glowroot.ui.sandbox.UiSandboxMain。</font><font style="vertical-align: inherit;">这将启动 Glowroot 并生成各种示例跟踪，以便为 UI 提供显示内容。</font><font style="vertical-align: inherit;">将浏览器连接到</font></font><a href="http://localhost:4000" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">http://localhost:4000</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">此外，Glowroot 使用</font></font><a href="http://bower.io" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Bower</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">和</font></font><a href="http://gruntjs.com" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">Grunt</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">构建其 Web 资产（依赖管理、Javascript concat/minify、LESS 编译/uncss/minify、AngularJS 模板 concat/minify、资产加速等）。</font><font style="vertical-align: inherit;">第一次运行时</font></font><code>mvn clean install</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，Node、Bower 和 Grunt 会本地安装在该</font></font><code>ui</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">目录下（感谢</font></font><a href="https://github.com/eirslett/frontend-maven-plugin"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">frontend-maven-plugin</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">）。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">如果您正在修改 Web 资产，则需要在每次更改后运行 grunt 来重新构建它们，或者（更好）</font></font><code>./grunt serve</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">从</font></font><code>ui</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">目录运行并将浏览器连接到</font></font><a href="http://localhost:9000" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">http://localhost:9000</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>
<p dir="auto"><code>./grunt serve</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">无需 concat/minify/rev 步骤即可将 Glowroot Web 资源提供给浏览器，这使得测试/调试变得更加容易。</font><font style="vertical-align: inherit;">它将非静态资源请求反向代理到</font></font><a href="http://localhost:4000" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">http://localhost:4000</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">，由 Glowroot 处理。</font><font style="vertical-align: inherit;">它还监视文件的更改并在浏览器内执行 Web 资源的实时重新加载。</font></font></p>
<p dir="auto"><code>./grunt serve:demo</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">与上面的操作相同，只是它将非静态资源请求反向代理到</font></font><a href="https://demo.glowroot.org" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">https://demo.glowroot.org</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">而不是</font></font><a href="http://localhost:4000" rel="nofollow"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">http://localhost:4000</font></font></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>
<h2 tabindex="-1" dir="auto"><a id="user-content-integration-tests" class="anchor" aria-hidden="true" tabindex="-1" href="#integration-tests"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">集成测试</font></font></h2>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">集成测试在 Maven 的标准生命周期阶段运行</font></font><code>integration-test</code><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">。</font></font></p>
